Condicionales de control de acción.

  • if(Op == "AVERAGE" || Op == "START")
  • if(Op == "VERBOSE")
  • if(Op == "MODE")
  • if(flag_verbose == "ON")
  • if(Op == "FDATA")

Ciclo while de ejecución de instrucciones.

  • while(Serial.available() == 0)

Dentro del ciclo while, hay condiconales de ejecución de funciones.

  • if(flag_start == true)
  • else if(flag_reset == true)
  • else if (flag_stop == true)
  • if(millis()). Se ejecuta siempre para la lectura de las variables y activación de funciones.

Intrucción. Lee una cadena de caracteres desde el serial.

  • Serial.readString();

Instrucción. Elimina los espacios en blanco y saltos de línea al inicio y al final del valor tomado en el serial.

  • Action.trim();

Función. Asigna la instrucción a la variable Op.

  • Op = Options(Action)

Cuando el programa inicia por primera vez, la BANDERA está desactivada, por lo que espera la Acción.

  • if(flag == false){}.

Activa la BANDERA para darle control total del sistema.

  • if(Op == "START" || Op == "RESET" || Op == "STOP"){}

Si no se cumple ninduna Acción y la bandera está desactivada, muestra el error y pide al ususario escribir START.

  • else{}

Cuando se activa la BANDERA entra al else, y se tiene el control del sistema.

  • else{Operations(Op);}
